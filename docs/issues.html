帮助 InPageEdit 完善

作为一个为编辑者造福的用户脚本，我们深知只有听取用户的意见、指导才能更加完善。

同时，作为一个正在发展、持续更新的项目，我们也深知使用过程中出现漏洞、问题是不可避的。

因此，我们采用了 Github issues 作为用户汇报意见的平台。

如何汇报？

进入 InPageEdit 的 Github repo 后，点击“Issues”，之后点击“Start a new issue”。

选择合适的模板，按提示详细说明你遇到的情况。

在您提交之后，机智的小鱼君或其他的志愿者其实根本没有会跟进此 issue。

由于人力有限以及缓存机制等问题，您提交的问题可能并不能立即得到修复，请耐心等待。
